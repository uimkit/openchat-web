# Openchat-WEB
企业级的 聚合聊天WEB客户端. 集成了 全球主流聊天软件. 包括:
- Whatsapp
- Telegram
- Messenger
- Wechat
- Wework(企业微信)
- 公众号
等.

我们所有产品 / 模块 均采用 Serverless, 云原生策略构建.
后端基于 iDaaS, aPaaS, iPaaS 等低代码 / 模块化 / 标准化接入.

便于 SaaS开发商 / 企业 / 开发者 快速搭建个性化的SaaS产品.




## 快速指南
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
